# how to develop with claude code

claude codeを起動するためのコマンド
- claude: 対話セッションを開始
- claude --continue: 直近のセッションを継続
- claude --resume: 過去のセッションから選択して再開

詳細：https://zenn.dev/hokuto_tech/articles/86d1edb33da61a
